# 附加功能 - 产品设计文档

## 定位

围绕"手机高效使用 Claude Code / OpenCode"目标的补充功能。

---

## 一、文件预览

### 场景
Claude Code 频繁生成/修改文件，手机上 `cat` 看代码体验差。

### 功能
- 终端输出中的文件路径可点击，点击后打开内置代码查看器
- 只读模式，不支持编辑
- 语法高亮（常见语言：Rust、JS/TS、Python、Go、HTML/CSS、JSON、YAML 等）
- 行号显示
- 支持横向滚动（长行不折行）
- 顶部显示文件名，支持复制路径

### 实现方式
- 前端识别终端输出中的文件路径（正则匹配）
- 点击后通过 WebSocket / IPC 请求服务端读取文件内容
- 服务端返回文件内容，前端用轻量高亮库渲染（如 highlight.js）

---

## 二、剪贴板增强

### 场景
手机上选择和复制终端内容非常困难，标准长按选择不适合终端文本。

### 功能
- **选择模式**：长按终端区域进入选择模式
- 进入后显示两个拖拽光标（起点/终点），可框选文本范围
- 选中区域高亮显示
- 底部弹出操作栏：复制 / 全选 / 取消
- 复制后自动退出选择模式，显示 toast 提示"已复制"

### 补充
- 双击快速选中一个单词
- 三击选中整行

---

## 三、连接管理

### 场景
用户可能有多台机器运行 xterm 服务端（如公司电脑、家里电脑、远程服务器），需要快速切换。

### 功能
- **服务器列表**：保存多个连接地址（名称 + IP:端口）
- 支持添加 / 编辑 / 删除
- 一键切换连接
- 显示连接状态（在线/离线）
- 记住上次使用的连接，下次打开自动连接

### 入口
- 顶栏左侧显示当前连接名称，点击展开服务器列表

### 存储
- 连接列表保存在客户端本地（localStorage / Tauri 本地存储）

---

## 四、命令收藏

### 场景
除了快捷输入栏的固定按钮，用户可能有一些不常用但很长的命令想保存下来。

### 功能
- 终端中长按一条历史输入 → 弹出"收藏"选项
- 设置中有"收藏命令"列表，可手动添加
- 收藏的命令可以：
  - 一键发送到终端
  - 编辑名称和内容
  - 删除
- 支持分组/标签（如"Git"、"Claude Code"、"部署"）

### 与快捷输入栏的区别
| | 快捷输入栏 | 命令收藏 |
|--|-----------|---------|
| 位置 | 键盘顶部常驻 | 设置/侧边栏中 |
| 数量 | 少量（5-8个） | 不限 |
| 用途 | 高频短命令 | 低频长命令 |
| 触发 | 一键点击 | 进入列表选择 |
