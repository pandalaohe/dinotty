# 设置功能 - 产品设计文档

## 定位

本项目是专为手机端使用 Claude Code / OpenCode 设计的 Web 终端客户端。设置功能让用户个性化终端外观。

---

## 设置入口

- 终端界面右上角**齿轮图标**（参考 Termux / iSH 等主流终端工具）
- 点击后从右侧滑出设置面板

---

## 一、主题设置

### 范围
整个 UI 统一跟随主题（终端区域 + 顶栏 + 按钮 + 设置面板）

### 预设主题
| 主题 | 说明 |
|------|------|
| Dark | 默认主题 |
| Light | 浅色 |
| Dracula | 紫色系经典 |
| Nord | 冷蓝色系 |
| Monokai | 暖色高对比 |
| Solarized | 护眼色系 |

### 自定义能力
用户可自定义以下颜色：
- 前景色（文字）
- 背景色
- 光标色
- ANSI 16 色

## 三、快捷输入

### 显示位置
虚拟键盘顶部固定一行**可横向滚动的快捷按钮栏**，无论当前处于哪个键盘面板都始终可见。

### 默认预设按钮

| 按钮名称 | 发送内容 | 自动回车 |
|----------|----------|----------|
| y | y | ✅ |
| n | n | ✅ |
| /compact | /compact | ✅ |
| /clear | /clear | ✅ |
| /cost | /cost | ✅ |
| /help | /help | ✅ |

### 自定义能力
每个快捷按钮可配置：
- **显示名称** — 按钮上展示的文字
- **发送内容** — 点击后实际发送到终端的字符串
- **自动回车** — 发送后是否自动附加回车（\r）

### 编辑方式（在设置面板中）
- 长按拖拽排序
- 左滑删除
- 点击加号添加新按钮

---

## 四、设置持久化

- 用户设置保存在服务端，关联用户/设备
- 切换设备或重新打开浏览器后设置自动恢复
