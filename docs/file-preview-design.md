# 文件预览面板设计

## 背景

用户在终端内工作时经常需要查看仓库内的图片、视频、脚本与文档，并在必要时向当前工程目录**导入/导出**文件。文件预览与 Web 预览共用同一套「预览面板」容器：浏览、`cwd` 内上传与下载。

## 与 Web 预览的一致性

- **容器**：分屏、比例拖拽、关闭/全屏等行为与 `web-preview-design.md` 一致。
- **入口位置**：与 Web 预览**同一工具栏区域**；用户点击「文件预览」入口后打开预览面板，**默认根目录为当前终端 Pane 的 `cwd`**，左侧展开该目录文件树。
- （补充）终端输出中的可识别路径点击可作为后续增强，**不作为 v1 必选入口**。

## 已定稿决策

| 项 | 结论 |
|----|------|
| 入口 | 工具栏与 Web 预览统一区域；点击后打开面板并加载**当前 `cwd`** |
| 仅选中文件夹 | 右侧**留白 + 简短提示**（不自动打开 README） |
| 目录范围 | **仅限当前 `cwd`**，不做多根工作区 |
| 窄屏 | 左侧文件树改为**抽屉**（可从边缘滑出或按钮唤起），右侧为内容区 |
| 布局风格 | **类 VS Code**：左文件树，右文件内容预览 |
| 上传 | 将设备上的文件写入 **当前 `cwd` 子树** 内指定目录（树选中文件夹为目标；未选中则落到 `cwd` 根） |
| 下载 | 将树中选中的**文件**导出到用户选择的本机路径（系统保存对话框）；目录不提供整包下载（首期） |

## 核心场景

| # | 场景 | 用户行为 | 期望体验 |
|---|------|---------|---------|
| 1 | 打开面板 | 工具栏点击文件预览 | 分屏出现预览面板；树从 `cwd` 根展开 |
| 2 | 浏览目录 | 在树中展开/折叠文件夹 | 树节点正常层级展示；当前选中文件夹高亮 |
| 3 | 仅目录选中 | 选中文件夹但不选文件 | **右侧空白 + 提示**（如「请选择文件以预览」） |
| 4 | 图片 / 视频 | 选中文件 | 右侧内容区展示媒体控件 |
| 5 | 代码 / 脚本 | 选中源码文件 | 右侧只读 + 语法高亮 |
| 6 | 文档 | md / txt / pdf（若支持）等 | 右侧渲染或内嵌查看 |
| 7 | 不支持类型 | 选中二进制等 | **不展示内容**，简短说明 |
| 8 | 上传文件 | 工具栏「上传」或拖入树节点（目标文件夹） | 多选文件；写入目标目录；**同名冲突**：二次确认覆盖或自动重命名（实现择一，产品偏好「自动重命名」更安全） |
| 9 | 下载文件 | 选中文件后工具栏「下载」或上下文菜单 | 弹出保存位置；大文件流式写入，失败可重试 |

## 布局（横屏 / 宽屏）

类 VS Code Explorer + 编辑器区：

```
┌─────────────────────────────────────────────────────────────┐
│  预览壳工具栏（路径 / 刷新 / 上传 / 下载 / 关闭 …，与 Web 预览壳对齐） │
├──────────────┬──────────────────────────────────────────────┤
│  文件树       │  内容预览区                                   │
│  （cwd 根）   │  （选中文件时显示；仅目录时留白+提示）          │
│              │                                              │
└──────────────┴──────────────────────────────────────────────┘
```

- 树与内容区之间的分隔条可拖拽调整宽度（最小树宽满足触控）。
- 树支持滚动；无搜索框为 v1 默认（可按后续版本增加）。

## 布局（竖屏 / 窄屏）

- **文件树**：默认收起，以**抽屉**呈现（例如左侧边缘手势或工具栏「目录」按钮打开）。
- **内容区**：占满主区域；打开抽屉时内容区可半透明遮罩或挤压（择一实现，以不遮挡当前预览阅读为准）。

## 支持预览的类型（首期）

| 类别 | 示例 | 预览方式 |
|------|------|----------|
| 图片 | png, jpg, gif, webp, svg, … | 适配缩放展示 |
| 视频 | mp4, webm, mov（视平台解码） | 内置播放器 |
| 代码 / 脚本 | 常见源码扩展名 | 只读 + 高亮 |
| 文档 | md, txt, log；pdf 若可行 | 渲染 / 文本 / PDF |

未列入且非可靠文本 → 不预览，仅提示。

**大文件**：文本超过阈值时截断预览并提示。

## 安全（Tauri / 后端）

- **读预览**：路径限制在**当前 Pane 的 `cwd` 子树内**，禁止路径穿越。
- **上传写入**：目标目录必须是 `cwd` 子树内的规范路径；拒绝 `..` 与符号链接跳出（若平台需额外校验则一并处理）。
- **下载**：仅从 `cwd` 子树内读源文件，写入由系统对话框确定的用户路径。
- 预览区仍**不执行**文件内容；写入仅发生在用户明确的上传/导出操作。

## 非目标（首期）

- 多文件夹工作区、Office 原生预览、压缩包内浏览。
- 文件夹打包下载、远端对象存储直传。
